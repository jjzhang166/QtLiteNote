from collections import deque
de = deque(range(5))
de.append(5) # 在最后(右边)添加元素5
de.appendlef(6) # 在左边添加6
de.pop() #移除右边的元素5
de.popleft() #移除左边元素6
de.rotate(3) #循环左移3次
de.rotate(-1) #循环右边1次

PEP8

pep8规范代码风格

  • pip install -U pep8 安装
  • pep8 --first code.py 显示最基本的信息
  • pep8 --first . 检查当前文件夹下所有py文件
  • pep8 --show-pep8 code.py 显示详细信息
  • pep8 --show-source --show-pep8 code.py 把不规范的源码也显示出来
  • pep8 --statistics -qq fold 显示一个文件夹中所有的代码情况


规范

  • 文件后面不能有空行
  • 顶层函数或类的前面得有两行空行
  • 行尾不能有空格 W291 trailing whitespace
  • 注释得以 '# '开头, E265 block comment should start with '# '. 这里得修改vim的插件nerdcommenter, 在NERD_commenter.vim 中搜索python，将其#后面加个空格即可
